
#+Title: Setting up NixOS, the easy way!

:NOTE: This setup works better with systemd-boot, be sure to edit boot.nix and/or hardware-configuration.nix if you're setting up a GRUB-based install

* NOTE
Use =show-trace= to get the error calling site.
#+BEGIN_SRC sh
nixos-rebuild build --flake ".#Krypton" --show-trace
#+END_SRC

* To build a VM image
#+BEGIN_SRC shell
nix build .#nixosConfigurations.vm.config.system.build.isoImage
#+END_SRC

* To install the VM image from an ISO
#+BEGIN_SRC fish
git clone https://gitlab.com/zstg/nixos-config ~/.nixos
sudo nix run nixpkgs#disko -- --mode destroy,format,mount ~/.nixos/hosts/vm/disko.nix
sudo nixos-install \
     # --root /mnt
     --flake ~/.nixos/#vm
#+END_SRC
